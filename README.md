## 📌 About The Project
Yeh ek **Console-based (ya GUI-based)** Java application hai jo banking operations ko simulate karta hai. Is project ko maine Core Java aur Database concepts ko deeply samajhne ke liye banaya hai.

## 🚀 Key Features
- **Account Opening:** Naya Savings ya Current account kholna.
- **Deposit & Withdraw:** Paise jama aur nikalna.
- **Fund Transfer:** Ek account se doosre account mein paise transfer karna.
- **Balance Inquiry:** Account balance check karna.
- **Transaction History:** Purani transactions (Passbook) dekhna.
- **Admin Module:** (Agar hai toh) Users ko manage karna.

## 🛠️ Built With
- **Java (JDK 8+)** - Core Logic
- **JDBC** - Database Connectivity
- **MySQL** - Data Storage
- **Swing (Optional)** - Frontend Framework
